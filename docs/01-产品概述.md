# 01 - 产品概述

## 1.1 产品定位

「Menu」是一款面向年轻人的菜单管理与采购清单生成工具。用户可编排每周菜单、维护菜品库，系统自动汇总生成采购清单并提供营养评估报告。

**一句话定位：** 帮你规划一周吃什么、买什么、营养够不够。

## 1.2 目标用户

- 自主做饭的年轻人（单人/情侣/小家庭）
- 希望规律规划饮食、减少每日决策负担的人群
- 需要根据菜单自动生成采购清单的人群

## 1.3 核心价值

| 价值点 | 说明 |
|--------|------|
| **降决策成本** | 每周一次编排菜单，零每日纠结 |
| **采购自动化** | 根据菜单和人数一键生成采购清单，无需手动统计 |
| **AI辅助配置** | LLM自动生成菜品食材配置和营养数据，减少初始化工作 |
| **营养可视化** | 基于周菜单产出营养摄入评估，异常指标提醒 |
| **跨平台** | iOS / Android / Web 三端覆盖 |

## 1.4 关键设计原则

1. **单用户优先** — 无需登录注册，开箱即用
2. **本地优先** — 核心数据存于本地SQLite，离线可用，后端仅提供AI能力和模板数据
3. **LLM收口** — LLM由管理员统一配置在服务端，用户无感知，仅使用AI能力

## 1.5 核心功能一览

```
菜单APP
├── 每周菜单管理 ──── 创建/编辑/删除 周菜单
│                     按天填写主食 + 菜品
│                     历史输入自动补全
├── 菜品库 ────────── 菜品 CRUD
│                     菜品关联食材配置（LLM生成 + 手动调整）
│                     支持自定义分类
├── 采购清单 ──────── 选菜单 + 人数 → 自动汇总食材
│                     导出 PDF / Excel
├── 营养报告 ──────── 基于周菜单 + LLM 产出评估报告
└── 设置 ──────────── 自定义分类管理、通用设置
```
