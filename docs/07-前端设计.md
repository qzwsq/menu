## 7.4 状态管理方案

采用 **Riverpod** 进行状态管理：

- **Provider**: 用于简单数据（如API配置、当前选中菜单ID）
- **StateNotifierProvider**: 用于复杂状态（如菜单编辑状态、营养报告状态）
- **FutureProvider**: 用于异步数据加载（如菜品列表、API调用结果）

### 核心 Provider 示例
```dart
// 当前编辑的菜单状态
final menuEditProvider = StateNotifierProvider<MenuEditNotifier, MenuEditState>((ref) {
  return MenuEditNotifier(ref);
});

// 菜品列表
final foodListProvider = FutureProvider<List<Food>>((ref) async {
  final dao = FoodDao();
  return dao.getAll();
});

// 分类列表
final categoryProvider = FutureProvider<List<Category>>((ref) async {
  final dao = CategoryDao();
  return dao.getAll();
});
```
